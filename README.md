## 🌍 Overview

**Albi Global** is a comprehensive, production-ready e-commerce marketplace platform designed to compete with Amazon, Alibaba, and Daraz. It provides a complete ecosystem for customers, sellers, delivery partners, and administrators with advanced AI features, multi-language support, and scalable architecture.

### Key Statistics
- **Modules**: 12+ Core Systems
- **AI Features**: 8+ Intelligent Services
- **Languages Supported**: 50+ Languages (via AI Translation)
- **Currencies**: Multi-currency Support
- **Security**: Enterprise-Grade Encryption & 2FA
- **Scalability**: Microservices-Ready Architecture

## 💰 Revenue System

### 3 Income Streams for Admin

#### 1. Seller Commission Wallet
- **Type**: Percentage-based commission on every sale
- **Percentage**: Admin-configurable (e.g., 5-15%)
- **Calculation**: Auto-calculated on order completion
- **Example**: 10% commission on ৳1000 order = ৳100 to Admin

#### 2. Business Seller Plan Wallet
- **Type**: Premium seller packages/subscriptions
- **Plans**:
  - Basic Plan: ৳5,000/month (100 products limit)
  - Professional Plan: ৳15,000/month (500 products limit)
  - Enterprise Plan: ৳50,000/month (Unlimited products)
- **Features Included**: Analytics, Advanced Tools, Priority Support

#### 3. AI Service Income Wallet
- **Type**: Usage-based pricing for AI features
- **Services**:
  - AI Product Recommendations: ৳0.10 per request
  - AI Translation Service: ৳0.05 per character
  - AI Chatbot Service: ৳0.20 per conversation
  - Smart Search Enhancement: ৳0.15 per search
- **Billing**: Monthly invoice to sellers

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- PostgreSQL 14+
- Redis 7+
- Docker & Docker Compose

### Installation

```bash
# Clone Repository
git clone https://github.com/syedsabbir2026-droid/Albi-Global.git
cd Albi-Global

# Install Dependencies
npm install

# Setup Environment
cp .env.example .env
# Edit .env with your credentials

# Start with Docker
docker-compose up -d

# Run Migrations
npm run migrate

# Seed Database
npm run seed

# Start Services
npm run dev
```

### Access Points
- **API**: http://localhost:3000
- **Admin Panel**: http://localhost:3001
- **Seller Dashboard**: http://localhost:3002
- **Mobile App**: Launch via npm run start:mobile
